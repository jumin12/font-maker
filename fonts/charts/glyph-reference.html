Eldaraure Glyph Reference

Eldaraure blends flowing Elvish (Tengwar-inspired) curves with angular Galactic Basic (Aurebesh-inspired) geometry. Type standard Latin keys; the font renders the custom glyphs shown below.

Number mapping: Keys 1–9 render digits one through nine. Key 0 renders ten (a combined 1+0 glyph).
